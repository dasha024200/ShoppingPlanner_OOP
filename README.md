Планер покупок (Shopping Planner)
Курсовий проєкт з дисципліни об’єктно-орієнтоване програмування.
Автор: Стернійчук Дарина, група ІПЗ-3/2.

Shopping Planner - це веб-додаток для створення списків покупок, керування товарами та контролю бюджету.
Користувач може додавати товари, встановлювати бюджет та відслідковувати загальну суму витрат.

Основні можливості
створення списків покупок;
додавання та видалення товарів;
встановлення бюджету;
категорії товарів;
сортування товарів;
експорт списку у CSV.

Технології
ASP.NET Core (.NET 8)
C#
Entity Framework Core
SQLite
MVC архітектура

У проєкті реалізовано кілька патернів проєктування:
Factory Method - створення об'єктів товарів;
Strategy - сортування елементів списку;
Observer - повідомлення про зміни бюджету.

Основні таблиці бази даних:
Users
ShoppingLists
ShoppingItems

База даних реалізована за допомогою SQLite та Entity Framework Core.
